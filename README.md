📚 Descrição
Este projeto consiste no desenvolvimento de um website moderno e responsivo para um restaurante de culinária mexicana. O objetivo é oferecer uma experiência digital imersiva que reflita a cultura vibrante do México, permitindo que os clientes visualizem o cardápio, conheçam a história do local e realizem reservas online.

🚀 Funcionalidades
- Menu Digital Interativo: Visualização de pratos categorizados (Tacos, Burritos, Enchiladas).
- Sistema de Reservas: Formulário funcional para agendamento de mesas.
- Design Responsivo: Adaptado para dispositivos móveis, tablets e desktops.
- Galeria de Fotos: Exibição dos pratos e da ambientação do restaurante.

🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
